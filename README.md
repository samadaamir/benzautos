Benz Autos is a modern and responsive automotive service website designed for car repair and maintenance businesses.
It showcases services, customer testimonials, brand partnerships, and provides a contact form for customers to schedule services.

🌐 Live Preview:
https://benzautos.tiiny.site

📌 Project Overview

Benz Autos is a single-page automotive service website built with HTML, CSS, and JavaScript.
The website highlights essential services such as engine repair, oil changes, tire services, and battery checks while maintaining a clean and user-friendly interface.

The goal of this project is to create a professional landing page for a car repair workshop that allows customers to easily explore services and get in touch.

✨ Features
🚘 Modern hero section with service call-to-action
🛠 Services section with visual service cards
ℹ️ About section describing the business
🏷 Brand slider showing partner brands
⭐ Customer testimonials slider
📞 Contact form for customer inquiries
📍 Business contact information
📱 Fully responsive navigation with hamburger menu
🔝 Back-to-top navigation in footer
